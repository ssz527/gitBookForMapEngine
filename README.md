# 开发指导

此文档是map地图引擎对外的api文档

## 使用方法
---

安装gitbook cli
```
    npm install -g gitbook-cli
```
clone 仓库
```
    git clone git@gitlab:root/map_service.git
```

编译
```
    gitbook build
```

serve支持查看命令

```
    gitbook serve
```
